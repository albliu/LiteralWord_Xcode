Gesture Controls
- Swipe Left : next chapter
- Swipe Right : previous chapter
- Pinch Zoom : change text size
- Long Click : Copy Text
* Android 2.2 does not support copying text beyond what the screen is displaying, even if seemingly it is scrolling. To select more text, change the font size to smaller
